This is for Descendant Selector (space)

The descendant selector matches all the elements that are descendents of a specified element

This is a span column so the styling isn't applied on it

This is for child selector (>)

The child selector selects all elements that are the children of a specified element

This is a span column so the styling isn't applied on it.

This is for Adjacent Sibling Selector (+)

The adjacent sibling selector selects all elements that are adjacent siblings of a specified element

This is a paragraph which is just outside the division so the styling is applied on it

But this is not immediately followed.
